# Nostr Resources for New Users

A curated guide to the Nostr ecosystem -- videos, podcasts, articles, and tools for newcomers joining the decentralized social protocol.

![Nostr booth at a conference, featuring purple branding, the Nostr ostrich logo, and planet-shaped decorations](https://grownostr.org/images/nostr-booth.jpg)

## What is this?

A single-page resource site that helps new users ("Nostriches") navigate the Nostr universe. No frameworks, no build tools -- just one HTML file with hand-crafted CSS carrying a dark, cosmic aesthetic with purple accents and glowing orbs.

## Sections

- **Videos** -- Talks and explainers from Nostr World, podcasts, and panels
- **Podcasts** -- Plebchain Radio, Thank God for Nostr, Soapbox Sessions, Nostr Compass
- **Articles & Guides** -- Deep dives from Lyn Alden, River, Grow Nostr, Soapbox, and Bitcoin Magazine
- **Tips & Hashtags** -- `#HOWDONOSTR` and `#asknostr` for community learning
- **Explore the Ecosystem** -- Nostr.how and Nostrapps directories

## Design

The site follows a circles-over-boxes philosophy. Rounded shapes, glowing orbs, and radial gradients give each section an organic, planetary feel -- every card floats in space rather than sitting in a rigid grid. The dark background with purple accent (`#a855f7`) channels a cosmic, arcane aesthetic.

- **Fonts:** Comfortaa (headings/nav), DM Sans (body)
- **Layout:** Responsive single-column with pill-shaped navigation
- **Cards:** Color-coded dots (red/video, green/podcast, blue/article, purple/ecosystem) with hover glow effects

## Running locally

Open `index.html` in a browser. That's it. No install, no build step.

## Contributing

Add resources by editing `index.html` directly. Follow the existing card markup pattern and pick the appropriate color-coded dot for the resource type.
